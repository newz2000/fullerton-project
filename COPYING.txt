This is the Fullerton project, a pre-configured software project that you 
customize allowing you to easily adopt front-end development best practices
without requiring you to install many dependencies.

It is made up of a number of projects, each governed by it's own separate
license (denoted below). The code that makes up the Fullerton project is 
provided vai the BSD license (also provided below).
